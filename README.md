# Atividade de DevOps

Repositório criado para demonstração e uso do Docker Compose em sala de aula.

Dentro da pasta do repositório, rode no terminal: `docker-compose up -d` para incializar os serviços.

Após isso, acesso o endereço:
Acesse o endereço `http://localhost:80`

Caso quiser visualizar o retorno do backend, acesse: `http://localhost:3001/usuarios`

Certifique-se de que não há outros containeres rodando nas portas **80**, e **3001**
